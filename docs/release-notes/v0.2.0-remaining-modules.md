# v0.2.0 — Remaining Disease Management Modules

## Summary

This release implements all remaining disease management modules (Phases 1–8)
as specified in the implementation plan. It adds 14 new DocTypes, 11 services,
5 portal pages, 11 reports, 2 desk pages, and comprehensive API layer.

## New DocTypes

| DocType | Module | Purpose |
|---------|--------|---------|
| Home Monitoring Entry | CDM Monitoring | Patient-entered and clinician-entered longitudinal readings |
| Diabetes Profile | CDM Enrollment | Disease-specific profile for diabetes enrollments |
| Obesity Profile | CDM Enrollment | Disease-specific profile for obesity enrollments |
| CDM Alert | CDM Protocols | Clinical and compliance alerts with rule engine |
| Medication Adherence Log | CDM Monitoring | Per-dose medication adherence tracking |
| Medication Side Effect Log | CDM Monitoring | Side effect reporting and tracking |
| Complication Screening Tracker | CDM Protocols | Protocol-driven screening due/overdue tracking |
| Care Gap | CDM Protocols | Care gap identification and lifecycle management |
| Diet Plan | CDM Care Plans | Clinician-prescribed dietary guidance |
| Meal Log | CDM Monitoring | Patient meal adherence logging |
| Patient Activity Log | CDM Monitoring | Patient activity/exercise logging |

| Care Coordinator Action | CDM Care Plans | Coordinator action documentation |

1. Create any missing CDM roles
2. Apply the permission matrix to all DocTypes
