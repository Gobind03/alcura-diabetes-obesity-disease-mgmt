# LLM Prompts

This directory stores prompts used for LLM-assisted development, code generation, and review tasks related to the CDM app.

## Usage

Store prompts as individual markdown files named descriptively:
- `enrollment-doctype-generation.md`
- `care-plan-validation-logic.md`
- `report-query-optimization.md`

Each prompt file should include:
- The prompt text
- Context/system instructions used
- Expected output format
- Notes on any manual adjustments made to LLM output
