# Reports Documentation

This directory documents all reports, dashboards, and analytics created by the CDM app.

## Planned Reports

| Report | Type | Module | Description |
|---|---|---|---|
| Enrollment Summary | Script Report | CDM Reports | Active enrollments by disease type, status, practitioner |
| Care Plan Compliance | Script Report | CDM Reports | Goal achievement rates, overdue interventions |
| Review Schedule | Query Report | CDM Reports | Upcoming and overdue periodic reviews |
| Patient Monitoring Dashboard | Dashboard | CDM Dashboards | Real-time vitals and lab trends for enrolled patients |
| Population Health Overview | Script Report | CDM Reports | Aggregate outcomes across the enrolled population |
| Alert Activity Log | Query Report | CDM Reports | Alert generation, resolution times, severity distribution |

## Documentation Template

Each report document should cover:
- Business purpose
- Filters / parameters
- Columns / data points
- Query logic or script summary
- Permissions (who can view)
- Sample output
