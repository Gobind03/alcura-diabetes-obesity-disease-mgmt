# User Flows

This directory contains step-by-step user flow documentation for CDM features.

## Planned Flows

- **Enrollment Flow**: Patient identification, eligibility assessment, enrollment creation
- **Care Plan Creation Flow**: Protocol selection, goal setting, intervention assignment
- **Periodic Review Flow**: Scheduling, conducting review via encounter, updating goals
- **Patient Self-Monitoring Flow**: Portal login, vitals submission, progress viewing
- **Alert Handling Flow**: Alert generation, clinician notification, acknowledgement, resolution
- **Protocol Application Flow**: Template selection, step customization, compliance tracking

Each flow document will include:
- Actor(s) involved
- Preconditions
- Step-by-step walkthrough with screenshots (when available)
- DocTypes and fields touched
- Post-conditions and side effects
